{% blocktranslate %}

This is Ur OSS World: a system of ecosystems collaboratively maintaining open source software, elongating the life span of packages and helping communities -- from entire programming languages to as small as web frameworks -- thrive without the fear of abandonment.

The purpose of Ur OSS World elongating package lifespan and compatibility with newer software is to make teams around the world more efficient, squashing fewer bugs, vendoring fewer packages, and upgrading fewer packages. As a collective, we are a global community of developers dividing the large task as a single developer into smaller, manageable chunks.

{% endblocktranslate %}

{% translate "History" %}

{% blocktranslate %}

Andrew Chen Wang was developing The Ur's iOS app; as he got started, he realized that maintenance of several iOS keychain wrappers were being left unmaintained with open PRs beginning to pile up.

Jazzband is a collective of Python (mostly Django) developers maintaining packages that are crucial to the Python (again, mostly Django) ecosystem such as django-oauth-toolkit, the only OAuth2-ready handler for native Django support compared to the generic oauthlib Python package used by several Python web frameworks.

As The Ur grows its metaverse, Andrew realized there was a need for a Jazzband for any kind of ecosystem. After a quick poll of approval on Reddit, The Ur decided to make an OSS system of maintainers like Jazzband but easily extensible for all types of ecosystems.
